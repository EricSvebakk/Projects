## A* maze-solver
A pygame-implementation of A*

### Controls

binding | description
:-------------------------:|:-------------------------:
Left-click | add walls
Spacebar | pause/unpause 
C | clear all walls
R | randomize walls
P | randomize start/end-point
Q | quit program
up | increase timestep
down | decrease timestep
